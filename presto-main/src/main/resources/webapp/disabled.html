Web Interface is Disabled

  • Insecure authentication over HTTP is disabled by default.
  • This could be overridden via "hetu.queryeditor-ui.allow-insecure-over-http" property of "etc/config.properties" (e.g. hetu.queryeditor-ui.allow-insecure-over-http=true).
  • Check openLooKeng Web Interface for details and user guides.
